# Changelog

All notable changes to Sash are documented here.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] — Unreleased

First public release.

### Added
- Keyboard window placement: left/right/top/bottom halves, four quarters,
  left/center/right thirds, left/right two-thirds, and maximize.
- Restore: return a window to its frame before the last placement.
- Width cycling: pressing the same left/right key steps `1/2 → 2/3 → 1/3`
  (toggleable in Settings).
- Move to previous/next display, or jump to a specific display (up to 6,
  `⌃⌥⌘`+1/2/3 with 4–6 assignable), preserving the window's relative size.
- Gaps: even spacing between windows and screen edges, adjustable in Settings.
- Menu bar app (no Dock icon) with the actions grouped by category.
- Launch at login.
- First-run onboarding for Accessibility permission, with auto-close once
  granted.
- In-app language switch (System / English / 日本語 / 한국어) that does not
  change the macOS system language. Full en/ja/ko localization.
- About tab showing name, version, description, the no-data-collection note,
  and a live memory/CPU readout for Sash itself.
- Placeholder app icon (generated by `Scripts/GenerateAppIcon.swift`).

### Notes
- Distributed directly (Developer ID signed + notarized) via GitHub Releases and
  Homebrew — not the Mac App Store (Accessibility control is incompatible with
  the required App Sandbox).
- Drag-to-snap is intentionally omitted in v1.0 (overlaps macOS native tiling).
